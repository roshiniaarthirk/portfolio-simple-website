# Personal Portfolio Website

A simple personal portfolio website built using **HTML** and **CSS**.  
It contains two pages: **Home** (showing profile, skills, interests, and projects) and **Contact** (to reach me).  
Designed with a clean layout that works well on different devices.

---

## Features
- Home page with profile, skills, and project overview
- Contact page to connect or send messages
- Clean design using basic HTML and CSS
- Responsive layout for different screen sizes

---

## Technologies Used
- **HTML5** – Structure of the pages  
- **CSS3** – Styling and layout  
- **Google Fonts** – Young Serif for typography
